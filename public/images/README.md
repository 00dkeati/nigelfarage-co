# Images for NigelFarage.co

## Current Placeholders
- `hero-placeholder.svg` - Hero banner background
- `portrait-placeholder.svg` - Portrait placeholder
- `reform-logo-placeholder.svg` - Reform UK logo placeholder

## Recommended Images to Add

### Hero Image
Replace `hero-placeholder.svg` with a high-quality hero image:
- Farage at a podium/rally
- Size: 1920x1080 or similar widescreen
- Name it: `hero.jpg` or `hero.webp`

### Portrait
Replace `portrait-placeholder.svg` with:
- Official headshot of Farage
- Size: 400x500 or similar portrait ratio
- Name it: `farage-portrait.jpg`

### Reform UK Logo
Replace `reform-logo-placeholder.svg` with:
- Official Reform UK logo
- Size: 200x80 or similar
- Name it: `reform-uk-logo.png`

### People Portraits
Add portraits for each person in /people:
- `nigel-farage.jpg`
- `richard-tice.jpg`
- `suella-braverman.jpg`
- etc.

## Image Sources (Free/CC Licensed)
- Wikimedia Commons: https://commons.wikimedia.org/wiki/Category:Nigel_Farage
- UK Parliament: Official MP portraits (public domain)
- Reform UK: May provide official assets

## After Adding Images
1. Update the code to reference new filenames
2. Run `npm run build` to verify
3. Deploy with `npx vercel --prod`
